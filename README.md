# **PRIME NUMBERS**

## INFORMATION

* **Program to find prime numbers from 1 to 100.**

## TECHNOLOGIES USED

* **JAVA**

## CONTENTS

* The variables **i**, **n**, **start** and **end** are defined with int.

* Printing prime numbers with a for loop.

## CODES

```Java
        import java.util.Scanner;
```

```Java

                for (i = start; i <= end; i++){

                    boolean isPrime = true;

                    for (n = 2; n <= (i / 2); n++) {

                        if (i % n == 0) {

                            isPrime = false;

                            break;
                        }
                    }

                    if (isPrime){

                        System.out.print(i + "-");
                    }
                }
            }
        }
```
